About This App

This app is dedicated to help our user be aware of COVID related info around them. Once you input the state abbreviation, it will return information including: state population, total cases and deaths, new cases and deaths, case density, vaccination info, ICU info and their ranking in the country.

For a better user experience, please check out abrreviation page and data definition page.

Thanks to Covidactnow.org for its amazing data.
